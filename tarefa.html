A diferença entre web e internet

Internet

A internet é a interconexão de redes de computadores, possibilitando a oferta de diversos serviços. É como se a internet fosse uma estrutura física e web um serviço que roda sobre ela. A web não é o único serviço possível na internet, existe email, FTP e outros.

WEB

A WEB é um serviço dentro da internet, que possui páginas que são interconectadas por links e acessadas em navegadores (chrome, explorer...), e funciona a base de protocolos HTTP.
